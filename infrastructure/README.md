# Infrastructure

Infrastructure as Code using modules from [devops-toolkit](https://github.com/susanavenda/devops-toolkit).

**Modules:**
- VPC with public/private subnets
- S3 bucket for static assets

**Usage:**
```bash
terraform init
terraform plan
terraform apply
```

See [devops-toolkit](https://github.com/susanavenda/devops-toolkit) for module documentation.
